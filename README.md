# Portfolio Risk Engine (R)

Built an R-based portfolio risk analytics engine using real global equity holdings.

## Overview

This project analyses a diversified portfolio of ASX and US equities using 10 years of historical market data.

It calculates:

* Annualised Return
* Annualised Volatility
* Sharpe Ratio
* Sortino Ratio
* Maximum Drawdown
* Historical VaR (95%, 99%)
* Monte Carlo VaR
* Expected Shortfall
* Correlation Matrix

## Portfolio Results

* Annual Return: 29.89%
* Volatility: 19.28%
* Sharpe Ratio: 1.45
* Max Drawdown: 18.04%
* Monte Carlo VaR (95%): -1.88%
