# Portfolio Risk Engine (R)

R-based portfolio analytics engine using real ASX + US holdings.

## Features

* 10 years of historical market data
* Sharpe Ratio / Sortino Ratio
* Maximum Drawdown
* Historical VaR
* Monte Carlo VaR
* Expected Shortfall
* Correlation Heatmap

## Example Results

* Annual Return: 29.89%
* Volatility: 19.28%
* Sharpe Ratio: 1.45
* Max Drawdown: 18.04%
