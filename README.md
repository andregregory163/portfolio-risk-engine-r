# Portfolio Risk Engine (R)

Built an R-based portfolio analytics engine using real market holdings.

## Features

* 10 years historical data via Yahoo Finance
* Portfolio annual return / volatility
* Sharpe Ratio
* Sortino Ratio
* Max Drawdown
* Historical VaR (95%, 99%)
* Monte Carlo VaR
* Expected Shortfall
* Correlation Heatmap
